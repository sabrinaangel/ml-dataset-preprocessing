# ML Dataset Preprocessing

Repository tugas mingguan mata kuliah **Machine Learning** — berisi eksplorasi data preprocessing dan implementasi berbagai algoritma machine learning per minggu perkuliahan.

## 📁 Struktur Repository

| Minggu | Topik | Isi |
|---|---|---|
| Week 2 | Data Preprocessing | Eksplorasi dan pembersihan dataset awal |
| Week 5-6 | Klasifikasi | Text Classification, K-Nearest Neighbors (KNN), Naive Bayes, Decision Tree |
| Week 7 | Artificial Neural Network | ANN untuk dataset tabular (`ann_tabularDatasets.ipynb`) |
| Week 8-9-10 | *(isi sesuai materi minggu tsb)* | *(lengkapi di sini)* |
| Week 11-12 | Regresi | Simple Linear Regression, Multivariate Linear Regression |
| Week 13-14 | Clustering | K-Means, Hierarchical Clustering |

## 🛠️ Tools & Library
- Python 3.x
- Jupyter Notebook
- pandas, numpy
- scikit-learn
- matplotlib / seaborn

## 🚀 Cara Menjalankan
1. Clone repository ini
```bash
   git clone https://github.com/sabrinaangel/ml-dataset-preprocessing.git
```
2. Install dependency yang dibutuhkan
```bash
   pip install -r requirements.txt
```
3. Buka notebook per minggu menggunakan Jupyter Notebook / VS Code
